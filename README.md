# Memorial – Audiodescrição (Projeto pronto)

Este pacote está pronto para subir no GitHub Pages. Inclui duas obras:

- **A Grande Onda de Kanagawa** — Katsushika Hokusai (c. 1829–1833) — `obra1-kanagawa.html` — áudio `media/Audio_Obra_01.mp3`
- **Elembo** — Johnson Muffaba (2025) — `obra2-elembo.html` — áudio `media/Audio_Obra_02.mp3`

## Como publicar
1. Crie um repositório público no GitHub e faça o upload de todos os arquivos desta pasta **na raiz**.
2. Vá em **Settings → Pages** e configure **Build and deployment → Deploy from a branch** na branch `main` (diretório `/`).
3. A URL pública ficará algo como `https://SEU-USUARIO.github.io/NOME-DO-REPO/`.
4. Gere QR Codes que apontem para:
   - `.../obra1-kanagawa.html`
   - `.../obra2-elembo.html`

## Onde colocar os áudios
- Coloque **Audio_Obra_01.mp3** e **Audio_Obra_02.mp3** dentro da pasta `media/`.

## Acessibilidade
- Páginas com hierarquia semântica adequada, controle de player, botões com alvo de toque ≥ 44px, contraste e “skip link”.
- Transcrição textual completa fornecida abaixo do player.
- Compatível com NVDA/VoiceOver.
